# Task 3 - User Management Frontend

## Description
Simple HTML, CSS, JavaScript frontend which is connect to Task 2  API. Users can be loaded, adedd, deleted.

## How to Run
1. Start backend server: `cd ../task2-rest-api && node index.js`
2. `index.html` open through live server
3. `Load Users` click button 

## Features
- Load Users from API
- Add new user
- Delete user
